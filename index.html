Sobre o DarkFeather

O DarkFeather é uma ferramenta gráfica desenvolvida em Python com Tkinter para visualizar e extrair informações detalhadas das redes Wi-Fi salvas no sistema Windows. Ideal para técnicos, profissionais de suporte ou curiosos sobre redes já conectadas.

Visualização de Redes

Veja todas as redes Wi-Fi salvas no Windows, com detalhes completos.

Extração de Senhas

Recupere a senha (ASCII e HEX), autenticação, criptografia e tipo de conexão.

Interface Moderna

Interface gráfica escura, intuitiva e com botões de cópia rápida.
